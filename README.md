# Customer Retention & Churn Analysis

## Objective
Analyze customer churn patterns and identify factors affecting retention.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
1. Churn rate was analyzed.
2. Subscription type influenced retention.
3. Contract length affected churn behavior.
4. Support calls were linked to churn.
5. Spending patterns differed between customer groups.

## Recommendations
1. Improve customer support.
2. Promote long-term contracts.
3. Reward loyal customers.
4. Identify high-risk customers early.
5. Strengthen retention strategies.
